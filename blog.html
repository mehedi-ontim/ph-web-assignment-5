Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM:

  • By Id: document.getElementById("Id");
  • By Class Name: document.getElementsByClassName("Class Name");
  • By Tag Name: document.getElementsByTagName("Tag Name");
  • querySelector: document.querySelector("CSS Selector");
  • querySelector: document.querySelectorAll("CSS Selector");

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML vs innerText vs textContent:

InnerHTML: Using innerHTML allows you to see exactly what’s in the HTML markup contained within a string, including elements like spacing, line breaks and formatting.
InnerText: This approximates the “rendered” text content of a node and is aware of styling and CSS. It’s most effective when you only need to see what’s in the element without the formatting.
TextContent: This retrieves and sets the content of the tag as plain text. It’s most effective when you want to see what’s in an element, plus styling.

Question-3: What is event delegation in the DOM?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM. It allows parent elements to respond to events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create, add, and remove elements dynamically using the DOM (Document Object Model).

1. Creating an Element

  • You can use the document.createElement() method to create a new HTML element.

2. Adding an Element

You can add the newly created element to the DOM using:
  • appendChild() (adds at the end of the parent).
  • append() (similar to appendChild, but allows multiple elements and text nodes).
  • innerHTML (not recommended for dynamic content due to security risks).

3. Removing an Element

You can remove elements using:

  • removeChild().
  • remove().
